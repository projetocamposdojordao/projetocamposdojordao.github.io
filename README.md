# Campos do Jordão - Jekyll Starter

Estrutura pronta para GitHub Pages + Jekyll

## Como publicar novos posts

1. Entre na pasta `_posts`
2. Crie um arquivo com nome assim:
   `2026-04-12-meu-novo-post.md`
3. Cole este modelo:

```md
---
layout: post
title: "Título do post"
description: "Descrição curta do conteúdo"
category: Notícias
---

Texto do post aqui.
```

## Categorias usadas

- Notícias
- Turismo
- Eventos
